Widget to get the latest vine video from twitter using hashtag.

== Description ==
<p>Vine: a mobile service that lets you capture and share short looping videos. Like Tweets, the brevity of videos on Vine (6 seconds or less) inspires creativity. Now that you can easily capture motion and sound. you can read more about vine at <a rel="nofollow" href="http://vine.co/blog">Vine Blog</a></p>
<p>This plugin used to fetch the latest vine video posted at twitter using hashtag.</p>
<p>For more plugins, themes and WordPress support , Please visit <a rel="nofollow" href="http://www.csschopper.com/">http://www.csschopper.com/</a></p>

== Installation ==

Extract the zip file and put latest-vine-video in wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
<p>Go to Appearance >> Widgets to get Latest Vine Video Widget </p>
<p><strong>Note: </strong>You have to use following format for the hashtag to be searched from twitter:- " #<tagname> vine " for example if you want to use dance as hashtag then use following syntax "#dance vine" in the "Hashtag" text box of widget. </p>
